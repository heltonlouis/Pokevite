# Pokevite

### Aprendizado:
Muito show de bola este mini curso oferecido pelo Yuri. 
Eu super recomendo o seu curso para quem queira aprender um pouco sobre Vue.js e Vite!

## Como rodar projeto?

```sh
git clone https://github.com/yurimarcon/PokeVite.git
```

```sh
cd PokeVite
```

```sh
npm i
```

```sh
npm run dev
```
